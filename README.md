# ClashToPlay
ClashToPlay.com is a personal project made by me and my companion Adrian, we both were experimenting how far artificial intelligence can go and asked ChatGPT to make the famous game "Clash Royale" it made a really simple game with 8 cards with infinite range, inmortal cards and so much more issues. The game wasnt playeble but it was fun and we saw that interesting and started changing things, adding troops, adding features, fixing bugs and so on untill I decided to buy a domain and publish the game so more people can try it. I purchesed a cheap domain (14$ at start and 1$ per month) and decidet using GitHub Pages to publish the new versions and update the game. At some point we changed our AI to Google AI Studio, a really advanced AI model for coding (it really uses Gemini but its especializes in coding). After some days we added a database to save player data and a login function that saves all the players data (username, password, trophies, wins, cards...) and with that we were able to add a leaderboard with real players on it. With all that made and like 2 months of working we finally made a really cool and playeble cart game, similar to Clash Royale but with other names AND NOT PAY TO WIN (the last was personal).
